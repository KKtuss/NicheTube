# NicheTube

A YouTube UI replica for thumbnail memes. Browse and discover meme thumbnails in a familiar YouTube-like interface.

## Features

- YouTube-inspired UI design
- Profile popup with contract address and social links
- Copyable contract address
- Responsive thumbnail grid
- Dark theme interface

## Deployment

This project is ready for deployment on Vercel. Simply connect your GitHub repository to Vercel and it will automatically deploy.

## Local Development

To run locally:

```bash
npx http-server -p 8000
```

Then open `http://localhost:8000` in your browser.

## Project Structure

```
├── index.html          # Main HTML file
├── styles.css          # Stylesheet
├── script.js           # JavaScript functionality
├── Buttons/           # Sidebar button icons
├── Fonts/             # Custom fonts
└── Thumbnails/        # Video thumbnails
```
